# Random Quote Generator

A simple Laravel project that generates and displays random quotes each time the page is refreshed. This project is designed to be quick and easy, making it perfect for practicing Laravel basics.

## Features

- Generates a random quote from a predefined list.
- Displays the quote on a clean, simple UI.
- Refresh the page to see a new random quote.
